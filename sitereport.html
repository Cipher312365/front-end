However, as I started working on more complex projects, I ran into a lot of issues with debugging my code. Development of My Site Over the Term

Over the course of the term, I developed a site that showcases my web development skills and experience. I focused on both the technical and design aspects of the site's user interface. For the design, I chose a clean and modern look, with a simple color scheme of black, white, and blue. I was inspired by sites such as CodePen and CSS-Tricks, which have a similar minimalistic design.

For the technical side of the site, I used HTML, CSS, and JavaScript to create a responsive and functional layout. I used media queries to ensure that the site looks good on all screen sizes, and implemented some basic JavaScript functions to add interactivity to the site.
